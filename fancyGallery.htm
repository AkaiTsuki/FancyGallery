Auto height

The height will be automatically set to the same value of item width. 

				$(#id).fancyGallery({numDisplay:6});

Custome height

Set the height of gallery as same as the image height. 

				$('#fancyGallery1').fancyGallery({numDisplay:4,height:80});

Maximum Height

Finding the maximum height of all item and set it as the gallery height. 

				$('#fancyGallery1').fancyGallery({numDisplay:4,height:'max'});
